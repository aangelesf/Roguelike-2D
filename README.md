# Roguelike 2D

A 2D roguelike game prototype made with C++ and raylib.

## Current Features

- Resizable game window
- Title screen
- Settings placeholder screen
- Character select screen
- Three placeholder characters:
  - Warrior
  - Mage
  - Rogue
- Character selection with left and right arrows
- Confirm button to enter gameplay
- Basic gameplay screen
- WASD player movement
- Selected character appears correctly in gameplay
- Quit confirmation popup on the title screen

## Tools Used

- C++
- raylib
- VS Code
- MSYS2 / MinGW
- Makefile
- Git / GitHub

## Planned Features

- Test room

- Enemies
- Weapons

This repository currently contains the first working prototype with menu navigation, character selection, and basic player movement.
